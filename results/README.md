# Results Directory

This directory is where all team members should save their experimental results.

## File Organization
- Save your results as CSV files here
- Use clear naming: `n10_results.csv`, `n100_results.csv`, `n1000_results.csv`, etc.
- Include columns for: n, iterations, execution_time, success, board_state
- Performance plots and graphs should be saved here
- Tables and analysis results should be stored here

## Usage
Person C should save experimental data here for final analysis and comparison tables.
Person D should save visualization outputs and figures here.

## Required Test Values
- n = 10
- n = 100
- n = 1000
- n = 10000
- n = 100000
- n = 1000000
